!gifth:

Shows the list of available commands

!giveaway:

Starts the Giveaway setup process by asking you certain questions.

!gifreroll:

Picks a new winner from the latest giveaway. If you want to reroll an older giveaway use !gifreroll [channel_name] [messageId].

!giftdel:

Cancels the most recent giveaway in the current channel. If you want to end an specific giveaway, use !giftdel [channel_name] [messageId].

!callme:

Make the bot call you anything. You can even make her call you Oniichan/Oneechan by using !callme [message].
